# 📊 SIOP Reporting Dashboard | Power BI

An interactive Power BI dashboard developed as part of the **Smith & Nephew Analytics Internship Assessment (Case Study 2 – SIOP Reporting)**. The project compares forecast snapshots with actual sales data to evaluate forecast performance using dynamic visualizations, KPI metrics, and business insights.

---

## 🚀 Features

- Compare multiple forecast snapshots
- Automatic Lag Classification
  - Resultant (Latest Snapshot)
  - Lag 0
  - Lag 1
- Revenue Forecast vs Actual Analysis
- Non-Revenue Forecast vs Actual Analysis
- CAPEX Forecast vs Actual Analysis
- Forecast Accuracy & Bias KPIs
- Interactive slicers for Country and Lag Type
- Dynamic Power BI dashboard

---

## 🛠️ Tech Stack

- Power BI
- Power Query
- DAX (Data Analysis Expressions)

---

## 📈 KPIs

- Revenue Actual Total
- Revenue Forecast Total
- Forecast Accuracy %
- Forecast Bias %
- CAPEX Accuracy %
- Non-Revenue Accuracy %
